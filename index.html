Featured Projects

AI Match Predictor

A machine learning project that predicts the outcome of EPL matches using real match data. I scraped data from FBref using requests and BeautifulSoup, cleaned it with pandas, and built predictive models using scikit-learn. The project includes model training, evaluation, and performance tuning to classify match outcomes based on historical stats.

View AI Match Predictor on GitHub

Pokémon Battle System

A turn-based combat simulator built in Python using object-oriented programming. Features include type-based strategy, damage calculation, trainer adoption logic, and multi-round battles with HP tracking. Includes modules for managing Pokémon, people, CSV data, and a dynamic CLI battle menu.


View Battle Simulator on GitHub

Rental Website

A full-stack rental management platform built with Flask, SQL, and HTML/CSS. Supports user login, dynamic inventory browsing, and a functional rental and return system with database integration.

InfoGuard

A UX research and design project focused on combating online misinformation. I conducted user interviews, developed personas, and designed a prototype that helps users recognize false information and reduce ideological polarization.
